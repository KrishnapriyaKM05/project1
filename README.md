#ROCK PAPER SCISSORS
A simple game of rock paper scissors which lets you compete with computer.

Tech Stack:html,css,javascript,python.

TO RUN LOCALLY:

git clone https://github.com/your-username/project1.git

cd project1

pip install -r requirements.txt

python app.py
